Números y Operaciones

Números reales, potencias, raíces, notación científica, proporcionalidad y porcentajes.

Conjuntos numéricos
ℕ Naturales
{0,1,2,3,…}
Contar objetos
ℤ Enteros
{…,−2,−1,0,1,2,…}
Temperaturas, deudas
ℚ Racionales
Fracciones p/q (q≠0)
Decimal exacto o periódico
ℝ Reales
ℚ ∪ irracionales
√2, π, e ∈ ℝ
Números irracionales
No pueden expresarse como fracción p/q. Su decimal es infinito y no periódico.
Ejemplos: √2 ≈ 1,41421…   √3 ≈ 1,73205…   π ≈ 3,14159…   e ≈ 2,71828…
Propiedad: √n es irracional si n no es cuadrado perfecto.
Potencias y sus propiedades
Operaciones con potencias (misma base)
an · am = an+m
an / am = an−m
(an)m = an·m
a0 = 1  (a ≠ 0)
a−n = 1/an
Potencias de productos y cocientes
(a·b)n = an·bn
(a/b)n = an/bn
a1/n = n√a
am/n = n√(am)
(√a)2 = a  (a≥0)
Raíces y radicales
Propiedades de raíces
√(a·b) = √a · √b
√(a/b) = √a / √b
n√(m√a) = n·m√a
Simplificación de radicales
Extrae factores que sean cuadrados perfectos.
√12 = √(4·3) = 2√3
Racionalización
Elimina raíces del denominador.
6/√3 = 6√3/3 = 2√3
1/(a+√b) → mult. por (a−√b)
Notación científica
Forma estándar
a × 10n    con    1 ≤ a < 10    y    n ∈ ℤ
3 700 000 = 3,7 × 106  |  0,000045 = 4,5 × 10−5
Multiplicar: (a×10n)·(b×10m) = (a·b)×10n+m
Proporcionalidad
Razón y proporción
Razón: cociente a/b.
Proporción: a/b = c/d ↔ a·d = b·c (propiedad fundamental).
Magnitudes proporcionales directas: y = k·x → al doble de x, doble de y.
Magnitudes proporcionales inversas: y = k/x → al doble de x, mitad de y.
Porcentajes
Cantidad = Base × (r/100)
Aumento del r%: nueva = base × (1 + r/100)
Descuento del r%: nueva = base × (1 − r/100)
Porcentajes sucesivos: ±r% y ±s% ≠ ±(r+s)%
Aplica factor total: (1±r/100)·(1±s/100)
Ejemplo resuelto 1 — Potencias
Simplifica y calcula: (23 · 42) / (82)
1
Expresa todo en base 2: 42=(22)2=24   82=(23)2=26
2
Numerador: 23·24 = 27
3
División: 27/26 = 21 = 2
Resultado: 2
Ejemplo resuelto 2 — Porcentajes sucesivos
Un artículo sube un 20% y luego baja un 20%. ¿Es el precio final igual al inicial?
1
Factor total: 1,20 × 0,80 = 0,96
2
El precio final es el 96% del inicial: ha bajado un 4%
No: el precio baja un 4% en total
Ejemplo resuelto 3 — Notación científica
Calcula (3,2 × 105) ÷ (8 × 102)
1
Divide las mantisas: 3,2 / 8 = 0,4
2
Divide las potencias: 105/102 = 103
3
0,4 × 103 = 4 × 102 (ajustar para que 1 ≤ mantisa < 10)
4 × 102
⚠ Errores típicos
• 23·24 = 27 (se suman exponentes, NO se multiplican).
• √(a+b) ≠ √a + √b   (error muy frecuente).
• Porcentajes sucesivos: +20% y −20% NO se anulan (ver ejemplo).
• En notación científica la mantisa debe estar en [1,10).

Álgebra

Polinomios, productos notables, factorización, Ruffini, ecuaciones (1.º, 2.º, bicuadradas, 3.º y 4.º grado) y sistemas.

Polinomios: operaciones
Definición
P(x) = anxn + an−1xn−1 + … + a1x + a0
Grado: mayor exponente con coeficiente ≠ 0.  Término independiente: a0.
Valor numérico P(a): sustituir x por a y calcular.
Suma/Resta
Agrupa términos semejantes (mismo grado).
(x2+2x)+(3x−1) = x2+5x−1
Multiplicación
Distributiva término a término.
(x+2)(x−3) = x2−x−6
División
P(x) = D(x)·C(x) + R
Usa división larga o Ruffini (si D es x−a).
Productos notables
Cuadrado de suma
(a+b)2 = a2+2ab+b2
Cuadrado de diferencia
(a−b)2 = a2−2ab+b2
Suma por diferencia
(a+b)(a−b) = a2−b2
Factorización de polinomios
Factor común
Extrae el MCD de todos los términos.
6x2+9x = 3x(2x+3)
Identidades notables
Reconoce cuadrados perfectos y diferencia de cuadrados.
x2−9 = (x+3)(x−3)
x2+6x+9 = (x+3)2
Trinomio x2+bx+c
Busca dos números cuya suma sea b y producto c.
x2+5x+6 = (x+2)(x+3)
Algoritmo de Ruffini
Si P(a)=0, entonces (x−a) es factor. Aplica Ruffini para dividir y reducir el grado.
Algoritmo de Ruffini — paso a paso
Cómo usar Ruffini
Objeto: dividir P(x) entre (x−a).
1. Escribe los coeficientes de P(x) ordenados por grado (pon 0 si falta alguno).
2. Escribe a a la izquierda.
3. Baja el primer coeficiente. Multíplica por a, súmale al siguiente coeficiente, y así sucesivamente.
4. El último número es el resto; los demás son los coeficientes del cociente (grado = grado P − 1).
Teorema del factor: si P(a)=0 (resto=0), entonces (x−a) divide exactamente a P(x).
Ejemplo: factoriza P(x) = x3−6x2+11x−6 con Ruffini
Factoriza completamente: x3 − 6x2 + 11x − 6
1
Candidatos a raíz racional: divisores de 6: ±1, ±2, ±3, ±6
2
P(1) = 1−6+11−6 = 0 → (x−1) es factor
3
Ruffini con a=1: coefs 1|−6|11|−6
                           ↓1|−5|6
                    1|−5|6|0 → cociente: x2−5x+6
4
Factoriza x2−5x+6: busca suma=−5 y producto=6 → −2 y −3 → (x−2)(x−3)
x3−6x2+11x−6 = (x−1)(x−2)(x−3)
Ejemplo: ecuación de 4.º grado con Ruffini
Resuelve: x4 − 5x2 + 4 = 0
1
Es bicuadrada. Cambio t = x2: t2−5t+4 = 0
2
Resolvemos: t = (5±√(25−16))/2 = (5±3)/2 → t1=4, t2=1
3
x2=4 → x=±2  |  x2=1 → x=±1
x = 2, −2, 1, −1
Ejemplo: ecuación de 3.º grado
Resuelve: x3 + x2 − 4x − 4 = 0
1
P(−1) = −1+1+4−4 = 0 → (x+1) es factor
2
Ruffini con a=−1: cociente x2−4
3
x2−4=0 → x=±2
x = −1, 2, −2
Ecuaciones de 2.º grado
Fórmula general
x = (−b ± √(b2−4ac)) / (2a)
ax2 + bx + c = 0 con a ≠ 0  |  Discriminante Δ = b2−4ac
Δ > 0
Dos soluciones reales distintas
Δ = 0
Una solución real doble
x = −b/(2a)
Δ < 0
Sin solución real
Ecuaciones bicuadradas
Forma y método
Forma: ax4 + bx2 + c = 0  (no aparece x3 ni x).
Método: cambio de variable t = x2 → ecuación cuadrática at2+bt+c=0.
Atención: t = x2 ≥ 0, así que se descartan soluciones t < 0.
Si t ≥ 0: x = ±√t (dos soluciones por cada t positivo; una si t=0).
Ejemplo: bicuadrada
Resuelve: 2x4 − 9x2 + 4 = 0
1
Cambio t = x2: 2t2−9t+4 = 0
2
Δ = 81−32 = 49 → t = (9±7)/4 → t1=4, t2=1/2
3
t1=4: x=±2  |  t2=1/2: x=±√(1/2) = ±√2/2
x = 2, −2, √2/2, −√2/2
Sistemas de ecuaciones lineales
Sustitución
Despeja una incógnita de una ecuación y sustituye en la otra.
Igualación
Despeja la misma incógnita en ambas ecuaciones e iguala los resultados.
Reducción
Opera las ecuaciones (suma o resta, multiplicando si es necesario) para eliminar una incógnita.
Sistema compatible determinado
Una sola solución. Las rectas se cortan en un punto.
Sistema compatible indeterminado
Infinitas soluciones. Las rectas son coincidentes.
Sistema incompatible
Sin solución. Las rectas son paralelas.
Ejemplo: sistema con método de reducción
Resuelve: { 3x + 2y = 12  |  2x − y = 1 }
1
Multiplica la 2.ª ecuación ×2: 4x−2y=2
2
Suma: (3x+2y)+(4x−2y)=12+2 → 7x=14 → x=2
3
Sustituye en 2·2−y=1 → y=3
x=2   y=3
Ejemplo: problema con sistema
La suma de dos números es 20 y su diferencia es 4. Hállalos.
1
Sistema: x+y=20  |  x−y=4
2
Suma: 2x=24 → x=12 → y=8
Los números son 12 y 8
⚠ Errores típicos
• (a+b)2 ≠ a2+b2: falta el término 2ab.
• Antes de Ruffini, siempre verificar P(a)=0.
• En bicuadradas, descartar t<0 antes de calcular las x.
• Al dividir una ecuación entera por un número, dividir TODOS los términos.

Funciones

Concepto, dominio, representación gráfica, función afin, cuadrática, inversa proporcional y análisis.

Concepto de función
Definición
Una función f: A → B asigna a cada x del dominio A un único valor f(x). Variable independiente: x. Variable dependiente: y = f(x).
Se puede definir por: tabla, fórmula algebraica, gráfica o a trozos.
Dominio
Valores de x donde f está definida.
f(x)=√x → x≥0
f(x)=1/x → x≠0
Recorrido (imagen)
Conjunto de todos los valores f(x) posibles.
Ceros (raíces)
Valores de x donde f(x)=0. Cortes con el eje X.
Características globales de una función
Crecimiento / Decrecimiento
Creciente en I: si x1<x2 → f(x1)<f(x2).
Decreciente en I: si x1<x2 → f(x1)>f(x2).
Constante: f(x)=k para todo x en I.
Máximos y mínimos relativos
Máximo relativo: punto más alto en su entorno.
Mínimo relativo: punto más bajo en su entorno.
En ese punto la función pasa de creciente a decreciente (o viceversa).
Signo de la función
f(x)>0: gráfica por encima del eje X.
f(x)<0: gráfica por debajo del eje X.
Cambia de signo en los ceros de f.
Continuidad
Una función es continua si su gráfica no tiene saltos ni agujeros. Si los tiene, es discontinua.
Función lineal y afín (recta)
Ecuación de la recta
y = mx + b
m = pendiente  |  b = ordenada en el origen (corte con eje Y)
Función lineal: b=0 → y=mx (pasa por el origen)
Pendiente
m = (y2−y1) / (x2−x1)
m>0 creciente · m<0 decreciente · m=0 constante
Rectas paralelas: misma pendiente. Perpendiculares: m1·m2=−1
Ecuación punto-pendiente
Si conocemos la pendiente m y un punto (x0,y0):
y − y0 = m · (x − x0)
Función cuadrática (parábola)
Parábola
y = ax2 + bx + c    (a ≠ 0)
a>0: abre hacia arriba (mínimo en el vértice)  |  a<0: abre hacia abajo (máximo)
Vértice: xv = −b/(2a)  |  Eje de simetría: x = −b/(2a)
Intersección con ejes
Eje Y: hacer x=0 → punto (0,c).
Eje X: resolver ax2+bx+c=0 (fórmula cuadrática).
El número de cortes depende del signo de Δ.
Función inversamente proporcional
y = k/x (k ≠ 0). Gráfica: hipérbola.
Dominio: x ≠ 0. Recorrido: y ≠ 0.
Si k>0: cuadrantes 1.º y 3.º. Si k<0: 2.º y 4.º
Funciones definidas a trozos
Definición
Se define con distintas fórmulas según el intervalo de x. Para evaluar, se sustituye x en la expresión del tramo correspondiente. Para representarla, se dibuja cada tramo solo en su intervalo.
Ejemplo resuelto 1 — Ecuación de la recta
Halla la ecuación de la recta perpendicular a y=2x+1 que pasa por el punto (4,3).
1
Pendiente de la recta dada: m=2. Pendiente perpendicular: m'=−1/2
2
Ecuación punto-pendiente con (4,3): y−3 = −1/2·(x−4)
3
Despejar: y = −x/2 + 2 + 3 = −x/2 + 5
y = −(1/2)x + 5
Ejemplo resuelto 2 — Análisis de parábola
Para f(x)=−x2+4x−3: halla vértice, ceros, signo y crecimiento.
1
a=−1<0 → parábola hacia abajo (máximo).
2
Vértice: xv=−4/(2·−1)=2   yv=−4+8−3=1 → V=(2,1)
3
Ceros: −x2+4x−3=0 → x2−4x+3=0 → (x−1)(x−3)=0 → x=1, x=3
4
Signo: f(x)>0 en (1,3); f(x)<0 en (−∞,1) ∪ (3,+∞)
5
Creciente en (−∞,2); decreciente en (2,+∞)
V=(2,1) · ceros x=1 y x=3 · máximo en x=2
Ejemplo resuelto 3 — Lectura de gráfica
De una gráfica sabemos: f es creciente en (−∞,2), tiene máximo en (2,5), es decreciente en (2,6), cero en x=6 y negativa en (6,+∞). Indica el dominio, cero, máximo y signo.
1
Dominio: todos los reales (o el intervalo observado en la gráfica)
2
Cero: x=6 (corte con eje X)
3
Máximo relativo: (2,5)
4
Signo: positiva antes de x=6, negativa después
Lectura correcta de todas las características
⚠ Errores típicos
• El vértice NO es (b,c): es xv=−b/(2a) y hay que calcular yv=f(xv).
• Rectas perpendiculares: m' = −1/m (no −m).
• Confundir cero (x donde f=0) con ordenada en el origen (f(0)).
• En funciones a trozos, usar la fórmula del intervalo correcto.

Geometría

Ángulos, figuras planas, teorema de Pitágoras, polígonos regulares, cuerpos geométricos y geometría espacial.

Teorema de Pitágoras
Teorema de Pitágoras
c2 = a2 + b2
c = hipotenusa (lado opuesto al ángulo recto)  |  a, b = catetos
Para verificar si es rectángulo: comprobar c2=a2+b2
Aplicaciones
• Hallar la diagonal de un rectángulo: d = √(l2+a2).
• Hallar la altura de un triángulo isósceles o equilátero.
• Calcular distancias en el plano: d = √(Δx2+Δy2).
• Triángulos pitaóricos clásicos: (3,4,5), (5,12,13), (8,15,17).

Sector circular

Generatriz del cono
La generatriz g es la distancia del vértice a cualquier punto de la circunferencia base.
g = √(r2 + h2) (Pitágoras: g es la hipotenusa, r y h son los catetos).
Cuerpos de revolución y secciones
Cilindro
Generado al girar un rectángulo alrededor de uno de sus lados.

Cono
Generado al girar un triángulo rectángulo alrededor de uno de sus catetos.
Sección transversal: círculo.
Esfera
Generada al girar una semicircunferencia alrededor de su diámetro.
Toda sección plana: círculo.

Vectores en el plano

Concepto, operaciones, módulo, punto medio, distancia y producto escalar.

Concepto de vector
Definición
Un vector es un segmento orientado con módulo (longitud), dirección y sentido. Se denota AB o bien v.
Vector con origen A(x1,y1) y extremo B(x2,y2):
AB = (x2−x1 , y2−y1)
Las componentes se calculan siempre como extremo menos origen.
Módulo
|v| = √(vx2 + vy2)
Longitud del vector. Siempre ≥ 0.
Vector unitario
Módulo = 1. Se obtiene dividiendo entre su módulo:
u0 = v / |v|
Vectores equipolentes
Dos vectores son iguales si tienen las mismas componentes (mismo módulo, dirección y sentido).
Operaciones con vectores
Suma
u+v = (ux+vx, uy+vy)
Regla del paralelogramo o triangulo
Resta
u−v = (ux−vx, uy−vy)
Producto por escalar k
k·v = (k·vx, k·vy)
Si k<0 cambia el sentido. Si k=0 vector nulo.
Distancia y punto medio
Distancia entre A y B
d(A,B) = √((x2−x1)2+(y2−y1)2)
Equivale al módulo del vector AB
Punto medio M de AB
M = ((x1+x2)/2 , (y1+y2)/2)
Producto escalar
Producto escalar (definición analítica)
u · v = ux·vx + uy·vy
También: u·v = |u|·|v|·cos(α)
Vectores perpendiculares: u·v=0  |  Vectores paralelos: u·v=±|u|·|v|
Condición de perpendicularidad
Dos vectores u=(a,b) y v=(c,d) son perpendiculares si y solo si:
a·c + b·d = 0
Condición de paralelismo
Dos vectores u=(a,b) y v=(c,d) son paralelos si y solo si:
a·d − b·c = 0
Aplicaciones: baricentro y repartión de segmentos
Baricentro de un triángulo
El baricentro G de un triángulo A(x1,y1), B(x2,y2), C(x3,y3) es:
G = ((x1+x2+x3)/3 , (y1+y2+y3)/3)
Es la intersección de las medianas.
Ejemplo resuelto 1 — Operaciones vectoriales
Dados u=(3,−1) y v=(2,4). Calcula u+v, 2u−v, |u| y el producto escalar.
1
u+v = (3+2, −1+4) = (5, 3)
2
2u−v = (6−2, −2−4) = (4, −6)
3
|u| = √(9+1) = √10
4
u·v = 3·2 + (−1)·4 = 6−4 = 2
u+v=(5,3) · 2u−v=(4,−6) · |u|=√10 · u·v=2
Ejemplo resuelto 2 — Ángulo entre vectores
Calcula el ángulo entre u=(1,0) y v=(1,1).
1
u·v = 1·1+0·1 = 1
2
|u|=1   |v|=√2
3
cos(α) = 1/(1·√2) = 1/√2 → α = 45°
El ángulo es 45°
Ejemplo resuelto 3 — Baricentro
Halla el baricentro del triángulo A(0,0), B(6,0), C(3,6).
1
Gx = (0+6+3)/3 = 9/3 = 3
2
Gy = (0+0+6)/3 = 2
G = (3, 2)
⚠ Errores típicos
• AB = B−A (extremo menos origen), NO A−B.
• El módulo siempre es positivo o cero.
• Perpendicularidad: producto escalar = 0 (no el módulo).
• El punto medio es la media de las coordenadas, no la diferencia.
Bloque 6 — Énfasis

Semejanza

Figuras semejantes, razón de semejanza, Teorema de Tales, criterios de semejanza y escalas.

Figuras semejantes
Semejanza
Dos figuras son semejantes si tienen la misma forma pero (posiblemente) distinto tamaño. Condiciones:
• Ángulos correspondientes iguales.
• Lados correspondientes proporcionales con la misma razón k (razón de semejanza).
Razón de semejanza
k = l' / l
l' = lado figura transformada, l = lado figura original
k > 1: ampliación  |  k < 1: reducción  |  k = 1: figuras congruentes
Relación de áreas y volúmenes
A'/A = k2     V'/V = k3
Si k=2: área ×4 y volumen ×8
Teorema de Tales
Enunciado
Si una recta es paralela a un lado de un triángulo y corta a los otros dos lados, divide esos lados en segmentos proporcionales.

En el triángulo ABC, si MN es paralela a BC (con M en AB y N en AC), entonces:
Proporción de Tales (forma 1)
AM / MB = AN / NC
Segmentos proporcionales en cada lado
Proporción de Tales (forma 2)
AM / AB = AN / AC
Segmento parcial sobre el lado completo
Lado cortado MN
MN / BC = AM / AB
El segmento paralelo es proporcional al lado de la base
Tales en un haz de rectas paralelas
Si varias rectas paralelas cortan a dos transversales, los segmentos que determinan en cada transversal son proporcionales entre sí.
Si a, b, c son paralelas que cortan a dos transversales en A,B,C y A',B',C' respectivamente:
AB / BC = A'B' / B'C'
Criterios de semejanza de triángulos
Criterio AA (o AAA)
Dos ángulos iguales en ambos triángulos. El tercero es automáticamente igual (suma de ángulos = 180°).
Criterio LAL
Un ángulo igual comprendido entre dos pares de lados proporcionales.
Criterio LLL
Los tres pares de lados correspondientes son proporcionales (misma razón k).
Escalas
Escala de un mapa o plano
E = distancia en el plano / distancia real
E = 1 : 50 000 significa: 1 cm en el mapa = 50 000 cm = 500 m en la realidad
Distancia real = distancia en mapa / E  |  Distancia en mapa = distancia real × E
Ejemplo resuelto 1 — Teorema de Tales
En el triángulo ABC, MN es paralela a BC. AB=10, AM=4, AC=15. Calcula AN, MB y NC.
1
MB = AB − AM = 10 − 4 = 6
2
Por Tales: AM/AB = AN/AC → 4/10 = AN/15 → AN = 15·4/10 = 6
3
NC = AC − AN = 15 − 6 = 9
4
Verificación: AM/MB = 4/6 = 2/3 y AN/NC = 6/9 = 2/3 ✓
AN=6 · MB=6 · NC=9
Ejemplo resuelto 2 — Razón de semejanza y áreas
Dos triángulos semejantes tienen lados 5 cm y 8 cm respectivamente. El triángulo menor tiene área 25 cm2. Calcula el área del mayor.
1
Razón de semejanza: k = 8/5
2
Razón de áreas: k2 = (8/5)2 = 64/25
3
A' = 25 · (64/25) = 64 cm2
A' = 64 cm2
Ejemplo resuelto 3 — Escala + Tales aplicado
Un árbol proyecta una sombra de 9 m. En el mismo momento, una estaca de 2 m proyecta una sombra de 3 m. ¿Cuánto mide el árbol?
1
Por semejanza (rayos de sol paralelos): hárbol/sombraárbol = hestaca/sombraestaca
2
h/9 = 2/3 → h = 9 · 2/3 = 6 m
El árbol mide 6 m
⚠ Errores típicos
• Las áreas escalan con k2, los volúmenes con k3: no multiplicar directamente por k.
• En Tales, el segmento parcial (AM) va sobre su lado completo (AB), no sobre el otro segmento (MB).
• Escala 1:n es reducción si n>1; es ampliación si n<1.

Estadística

Variables, tablas de frecuencias, medidas de centralización y dispersión, e interpretación gráfica.

Variables estadísticas
Cualitativa
Expresa una cualidad, no numérica.
Color de ojos, deporte favorito, tipo de película
Cuantitativa
Discreta: valores aislados (número de hijos, goles).
Continua: valores en un intervalo (altura, temperatura).

TipoNotaciónDefiniciónPropiedad
Frecuencia absolutafiN.º de veces que aparece el dato xiΣfi = N (total)
Frecuencia relativahifi / N   (proporción)Σhi = 1
Frec. absoluta acumuladaFiSuma de fj hasta xi (inclusive)Fn = N

Datos agrupados en intervalos
Cuando los datos son muy numerosos o continuos, se agrupan en intervalos [a, b).
Marca de clase xi: punto medio del intervalo → xi = (a+b)/2.
Amplitud del intervalo: c = b−a. Todos los intervalos suelen tener la misma amplitud.
Medidas de centralización
Media aritmética
x̄ = (Σ fi·xi) / N
N = Σfi = total de datos
Mediana (Me)
Valor central al ordenar los datos.
• N impar: el dato central.
• N par: media de los dos centrales.
En tabla: el dato cuya Fi supera N/2 por primera vez.
Moda (Mo)
Dato con mayor frecuencia absoluta fi.
Puede haber más de una moda (bimodal) o ninguna.
En intervalos: intervalo modal = el de mayor frecuencia.
Medidas de dispersión
Varianza
σ2 = (Σ fi·(xi−x̄)2) / N
Siempre ≥ 0. Si todos los datos son iguales, σ2=0.
Desviación típica
σ = √(σ2)
Mismas unidades que los datos.
Mide la dispersión respecto a la media.
Recorrido (rango)
R = xmáx − xmín

Medida simple de dispersión. Sensible a valores extremos.
Gráficos estadísticos
Diagrama de barras
Variables discretas o cualitativas. Cada barra es proporcional a fi.
Histograma
Variables continuas agrupadas. Las barras son adyacentes.
Diagrama sectorial
Proporciones. Ángulo = hi × 360°
Polígono de frecuencias
Une los puntos medios de las barras del histograma.

Probabilidad

Espacio muestral, sucesos, Laplace, probabilidad compuesta e independencia.

Espacio muestral y sucesos
Experimento aleatorio y espacio muestral
Experimento aleatorio: resultado no predecible (dado, carta, urna).
Espacio muestral E: conjunto de todos los posibles resultados.
Dado de 6 caras: E = {1,2,3,4,5,6} → 6 sucesos elementales.
Suceso elemental
Un solo resultado. {3}
Suceso compuesto
Varios resultados. {2,4,6}
Suceso seguro
P=1. Ocurre siempre. E completo.
Suceso imposible
P=0. Nunca ocurre. Conjunto vacío.
Operaciones con sucesos
Unión A ∪ B
Ocurre A o B (o ambos). Todos los resultados de A y B.
Intersección A ∩ B
Ocurren A y B a la vez. Solo los resultados comunes.
Suceso contrario Â
Todo lo que NO es A. P(Â) = 1 − P(A).
Sucesos compatibles e incompatibles
Compatibles: pueden ocurrir a la vez (A ∩ B ≠ ∅).
Incompatibles (mutuamente excluyentes): no pueden ocurrir a la vez (A ∩ B = ∅).
Ejemplos: par e impar son incompatibles en un dado.
Sucesos contrarios
Son incompatibles y juntos forman el espacio muestral:
A ∪ Â = E   y   A ∩ Â = ∅
Ejemplo: sacar par / sacar impar en un dado.
Regla de Laplace
Regla de Laplace (equiprobabilidad)
P(A) = casos favorables / casos posibles totales
Válida cuando todos los resultados son equiprobables  |  0 ≤ P(A) ≤ 1
Suceso contrario
P(Â) = 1 − P(A)
Probabilidad compuesta
Unión (fórmula general)
P(A∪B) = P(A)+P(B)−P(A∩B)
Si A y B son incompatibles: P(A∪B)=P(A)+P(B)
Intersección (sucesos independientes)
P(A∩B) = P(A) · P(B)
Solo válida si A y B son independientes
Sucesos independientes
El resultado de uno NO afecta al del otro.
Ejemplos: lanzar dos dados, sacar una carta y volverla a meter.
Sucesos dependientes
El resultado de uno SÍ influye en el del otro.
Ejemplo: extraer bolas sin reposición (el espacio muestral cambia).
Probabilidad experimental
Frecuencia relativa como probabilidad experimental
Pexp(A) = fi(A) / n
fi(A) = n.º de veces que ocurrió A  |  n = n.º de repeticiones
Ley de los grandes números: cuando n → ∞, Pexp → Pteórica
Ejemplo resuelto 1 — Unión con sucesos compatibles
Dado de 6 caras. A = número par; B = número mayor que 3. Calcula P(A∪B).
1
A={2,4,6} → P(A)=3/6=1/2. B={4,5,6} → P(B)=3/6=1/2
2
A∩B={4,6} → P(A∩B)=2/6=1/3
3
P(A∪B)=1/2+1/2−1/3=3/6+3/6−2/6=4/6=2/3
P(A∪B) = 2/3
Ejemplo resuelto 2 — Sin reposición (dependientes)
Urna con 4 bolas rojas y 3 azules. Se extraen 2 sin reposición. P(las dos rojas) y P(una de cada color).
1
P(2 rojas) = P(1.ª roja) · P(2.ª roja | 1.ª roja) = 4/7 · 3/6 = 12/42 = 2/7
2
P(roja 1.ª, azul 2.ª) = 4/7 · 3/6 = 2/7   P(azul 1.ª, roja 2.ª) = 3/7 · 4/6 = 2/7
3
P(una de cada) = 2/7 + 2/7 = 4/7
P(2 rojas)=2/7 · P(una de cada)=4/7
